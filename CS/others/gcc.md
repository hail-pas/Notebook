# GCC

## 参数
```shell
-ggdb
-Wall                         # 全部的警告
-fdiagnostics-color=always    # 输出彩色日志
-fsanitize=address            # 動態程序分析, 地址訪問檢查
-fsanitize=thread             # 綫程 data race
-fsanitize=memory             # memory
-fsanitize=ub                 # undefined
-O2                           # 優化

```
